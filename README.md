# LvDReference
This is a reference implementation for the LvD algorithm.

The algorithm is describe in
https://arxiv.org/abs/2601.19064

This implementation is used to illustrate the algorithm and to make the simulation benchmarks in the paper.

It has not been optimized to any great extent.


## Repository Structure

This repository contains reference implementations of the LvD likelihood algorithm and associated tools, developed as part of ongoing research.

- **`main`** — current development version of the codebase.
- **`firstSubmissionExperiments`** — snapshot of the code and experiments associated with the first submission of the paper. Use this branch to reproduce the results reported in the initial submission.

For reproducibility of the results in the initial paper, please check out the `firstSubmissionExperiments` branch:

```
git checkout firstSubmissionExperiments
```
